# Aioni Digital and Data Solutions Inc.

**AS153174** — Enterprise Internet & Network Solutions, Philippines

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy

```bash
npm run deploy
```

## Edit Content

All content is in `data/*.json`. Edit and run `npm run deploy` to push changes live.

To add coverage cities, edit `data/coverage.json` and run `npm run build:map` first.
